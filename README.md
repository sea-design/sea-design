# Sea Design

🌊 An UI design language and React UI library.

## ⌨️ Development

This will open a development server on port 6006.

```bash
$ git clone git@github.com:sea-design/sea-design.git
$ cd sea-design
$ yarn
$ yarn start
```

## 📖 Doc Generation

```bash
$ yarn docs
```

Commit `docs` folder and view site https://sea-design.github.io/
